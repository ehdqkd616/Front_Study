배열

자바스크립트에서는 자료형 지정이 없기 때문에 한 배열 안에 모든 자료형 보관 가능(컬렉션과 비슷)


배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 크기를 정하고 선언할 수도 있음


배열 메소드

배열도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드가 있음

indexOf()

배열에서 요소가 위치한 인덱스 리턴


join()

배열을 결합하여 문자열로 반환


sort()

배열 내림차순 혹은 오름차순 정렬


push()와 pop()

push() : 배열 맨 뒤에 요소 추가
pop() : 배열 맨 뒤 요소 제거


shift() 와 unshift()

shift() : 배열의 맨 앞에 요소 제거
unshift() : 배열의 맨 앞에 요소 추가


slice() 와 splice()

slice() : 배열의 요소 뽑아내기
splice() : 배열의 index위치의 요소 제거 및 추가


toString()

배열을 문자열로 반환
